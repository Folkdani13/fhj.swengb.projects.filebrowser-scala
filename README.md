# fhj.swengb.projects.filebrowser-scala


## How to use this repository
1. Fork
2. Clone your Fork
3. In IntelliJ, right click the "pom.xml" that is within the project directory and select "+ Add as Maven Project" at the bottom of the list. (In case you ever need to undo this step and remove the *.iml file that is generated by IntelliJ, go to "File -> Project Structure", select "filebrowser-scala" and press the "-" button to remove it.
